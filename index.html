Stripe Setup (Fast)

  1. Create a Stripe account.
  2. Go to Payment Links → Create a link for each product.
  3. Copy each payment link URL.
  4. Open app.js and paste the URLs into STRIPE_LINKS.
  5. Done — your “Buy Now” + “Checkout” will work.
